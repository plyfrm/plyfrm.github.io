BMSSearch Artist Table

Add this page's URL to BeMusicSeeker or beatoraja to use the table.

This is a difficulty table separating artists each into their own folder, generated using the BMSSearch API. Note that it definitely isn't perfect given the data on BMSSearch isn't always reliable to begin with, as well as there being no API endpoint for querying an artist's BMS meaning I had to use the search endpoint and filter out as many junk results as I could. Beatoraja also cannot have the same BMS in multiple folders so collaborations will only be present on one of the participants' folder (I believe this works properly in BeMusicSeeker/LR2).
